Class Manager

TFC manager represent the TFCManager smart contract, which serves as a proxy for TFC users to claim (withdraw) TFC tokens.

claimTFC

  • claimTFC(amount: BN, nonce: BN, sig: string, claimer: Account): Promise<void>
    • Defined in Manager.ts:80

    Claim TFC tokens with the authorization from signer. This is the function called by users who want to claim/withdraw TFC tokens.

    Parameters

    • amount: BN

      the amount of TFC token to claim

    • nonce: BN

      the nonce used in the signature

    • sig: string

      the signature signed by the signer who has privilege to authorize TFC claim

    • claimer: Account

      the account to receive the claimed TFC tokens. It must be the same with the recipient of signature

    Returns Promise<void>
