Class TFC

TFC Token contract representation.

Objects of this class can optionally set a default account, which will be used to send Ethereum transactions.

transferFrom

  • transferFrom(from: Address, to: Address, amount: BN, sender?: Account): Promise<void>
    • Defined in TFC.ts:170

    Transfer money from account {@param from} to account {@param to}. This is useful when an agent transfers tokens as a delegate of a user. The transaction sender must have enough allowance from account {@param from} to spend tokens.

    Parameters

    • from: Address

      Ethereum address of the account whose tokens are transferred out

    • to: Address

      Ethereum address of the account who receives the tokens

    • amount: BN

      the amount to transfer

    • Optional sender: Account

      (optional) transaction sender. If omitted, use the default account. This sender is different from the {@param from}.

    Returns Promise<void>
